> 대학 야구 동아리 대회 **H조 순위와 본선 진출 경우의 수**를 직접 계산하고 확인할 수 있도록 만든 Streamlit 기반 미니 프로젝트입니다.  
> 현재 순위, 잔여 경기, 예상 순위 변화, 팀별 상위 2위 가능 시나리오를 한 화면에서 볼 수 있도록 구성했습니다.

---

## 📌 프로젝트 소개

이 프로젝트는 대학 야구 동아리 대회에서  
제가 속한 팀의 **진출 경우의 수를 직접 계산해보기 위해 만든 Streamlit 대시보드**입니다.

처음에는 단순히 현재 순위와 남은 경기만 정리한 정적 화면이었지만,  
업그레이드하면서 사용자가 **남은 경기 결과를 직접 선택**하고,  
그에 따라 **최종 순위가 어떻게 바뀌는지** 바로 확인할 수 있도록 개선했습니다.

즉, 단순한 순위표가 아니라

- 현재 순위 확인
- 잔여 경기 결과 선택
- 예상 순위 자동 반영
- 특정 팀 기준 상위 2위 가능성 계산
- 전체 시나리오 표 확인

까지 가능한 **경우의 수 시뮬레이터** 형태로 만든 프로젝트입니다.

---

## 🚀 주요 기능

### 1. 현재 순위표 표시
- 팀별 승/패
- 경기 수
- 승률
- 득실차(가정)

를 표 형태로 한눈에 확인할 수 있습니다.

### 2. 잔여 경기 표시
남아 있는 경기 일정을 카드 형태로 보여주어  
남은 대진을 쉽게 파악할 수 있습니다.

### 3. 경기 결과 선택 시뮬레이션
각 경기마다 승리 팀을 직접 선택할 수 있고,  
점수 차까지 입력할 수 있어 득실차(가정) 반영이 가능합니다.

### 4. 예상 순위 자동 반영
사용자가 선택한 결과를 기준으로  
현재 예상 순위표가 바로 갱신됩니다.

### 5. 팀별 상위 2위 가능성 계산
특정 팀을 선택하면  
남은 경우의 수 전체를 계산해

- 상위 2위 가능 횟수
- 1위 가능 횟수
- 2위 가능 횟수
- 전체 경우의 수 대비 비율

을 자동으로 보여줍니다.

### 6. 전체 시나리오 표 제공
남은 경기의 가능한 시나리오들을 모두 정리해서  
각 팀이 몇 위로 끝나는지 표로 확인할 수 있습니다.

### 7. 유리 / 불리 시나리오 분리
선택한 팀 기준으로

- 상위 2위 진입 시나리오
- 탈락 시나리오

를 따로 볼 수 있도록 구성했습니다.

## 🔧 구현 내용

### 1. 순위 데이터 구조화
팀별 승/패/득실차를 딕셔너리 형태로 관리하고,  
이를 바탕으로 DataFrame을 생성해 순위표를 구성했습니다.

### 2. 승률 및 순위 계산
각 팀의 경기 수와 승률을 계산하고,  
승수 → 득실차(가정) → 승률 → 팀명 순으로 정렬해 순위를 계산하도록 구현했습니다.

### 3. 경기 결과 반영 함수 구현
각 경기의 승리 팀과 점수 차를 받아  
승/패와 득실차를 상태값에 반영하는 함수를 따로 구성했습니다.

### 4. 경우의 수 전체 탐색
아직 결과를 선택하지 않은 경기들에 대해  
가능한 모든 조합을 `itertools.product`로 생성해  
전체 시나리오를 자동 계산하도록 구현했습니다.

### 5. 특정 팀 기준 결과 요약
선택한 팀이

- 1위로 끝나는 경우
- 2위로 끝나는 경우
- 상위 2위 안에 드는 경우
- 탈락하는 경우

를 집계해 요약 지표로 보여주도록 만들었습니다.

### 6. Streamlit UI 구성
탭 구조, 카드형 경기 표시, 팀 선택, 데이터프레임 출력 등을 활용해  
복잡한 경우의 수를 최대한 보기 쉽게 정리했습니다.
